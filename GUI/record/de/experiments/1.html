📚 Theory

What is a Logic Gate?

A logic gate is the basic building block of digital circuits that implement Boolean functions.

Inputs: 0 = LOW & 1 = HIGH

Types of Logic Gates:

AND Gate
Y = A • B
Output is HIGH only when all inputs are HIGH
NAND Gate
Y = (A • B)'
Output is LOW only when all inputs are HIGH
OR Gate
Y = A + B
Output is HIGH when at least one input is HIGH
NOR Gate
Y = (A + B)'
Output is HIGH only when all inputs are LOW
NOT Gate
Y = A'
Output is the complement of the input
XOR Gate
Y = (A' • B) + (A • B')
Output is HIGH when inputs are different
X-NOR Gate
Y = (A • B) + (A' • B')
Output is HIGH when inputs are same

⚡ Procedure

  1. Connect the 5V DC power supply to the digital trainer kit
  2. Insert the respective IC for each logic gate on the breadboard
  3. Connect the input pins to the logic switches on the trainer kit
  4. Connect the output pin to the LED indicator
  5. Apply different input combinations (00, 01, 10, 11) for two-input gates
  6. Apply input combinations (0, 1) for the NOT gate
  7. Observe and record the output for each input combination
  8. Verify the results with the theoretical truth tables

✅ Conclusion

Successfully studied and verified the truth tables of all basic logic gates (AND, NAND, OR, NOR, NOT, XOR, X-NOR). The practical implementation confirmed the theoretical Boolean operations for each gate. These fundamental gates serve as building blocks for more complex digital circuits and are essential for understanding digital electronics and computer organization.
