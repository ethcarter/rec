Experiment 07

String Manipulation Functions

C++ Programming

üéØ Aim of the Experiment

Write a program to demonstrate use of string manipulation functions in C++

üíª Program

C++ 
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string str = "Hello World";
    
    cout << "Original string: " << str << endl;
    cout << "Length: " << str.length() << endl;
    cout << "Uppercase: ";
    
    string upper = str;
    transform(upper.begin(), upper.end(), upper.begin(), ::toupper);
    cout << upper << endl;
    
    cout << "Substring (0,5): " << str.substr(0, 5) << endl;
    cout << "Find 'World': " << str.find("World") << endl;
    
    str.replace(6, 5, "C++");
    cout << "After replace: " << str << endl;
    
    return 0;
}

üìã Sample Output

Output 
Original string: Hello World
Length: 11
Uppercase: HELLO WORLD
Substring (0,5): Hello
Find 'World': 6
After replace: Hello C++

üìö Key Concepts Demonstrated

  • String Class: Using C++ string class for dynamic string operations
  • Length Function: Getting string length with str.length()
  • Case Conversion: Using transform() with ::toupper for case conversion
  • Substring Extraction: Using substr() to extract portions of strings
  • Search Operations: Using find() to locate substrings
  • String Replacement: Using replace() to modify string content
  • Algorithm Header: Utilizing STL algorithms for string processing

‚úÖ Conclusion

String manipulation functions in C++ provide powerful tools for text processing operations. The program successfully demonstrates essential string operations including length calculation, case conversion, substring extraction, searching, and replacement.
