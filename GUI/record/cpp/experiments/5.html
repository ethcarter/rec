Experiment 05

Loop Statements

C++ Programming

üéØ Aim of the Experiment

Write a program using Loop statements in C++

üíª Program

C++ 
#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "Enter a number: ";
    cin >> n;

    // For loop
    cout << "\nUsing for loop:\n";
    for (int i = 1; i <= n; i++) {
        cout << i << " ";
    }
    cout << endl;

    // While loop
    cout << "\nUsing while loop:\n";
    int i = 1;
    while (i <= n) {
        cout << i << " ";
        i++;
    }
    cout << endl;

    // Do-while loop
    cout << "\nUsing do-while loop:\n";
    i = 1;
    if (n >= 1) {
        do {
            cout << i << " ";
            i++;
        } while (i <= n);
    }
    cout << endl;

    return 0;
}

üìö Key Concepts Demonstrated

  • For Loop: Entry-controlled loop with initialization, condition, and increment in one line
  • While Loop: Entry-controlled loop that checks condition before execution
  • Do-While Loop: Exit-controlled loop that executes at least once before checking condition
  • Loop Variables: Managing loop counters and their scope
  • Conditional Safety: Adding checks to prevent infinite loops or invalid operations
  • Output Formatting: Controlling display format within loops

‚úÖ Conclusion

A program was successfully written to demonstrate loop statements (for, while, do-while) in C++. Each loop type serves different purposes: for loops are ideal for known iterations, while loops for condition-based repetition, and do-while loops when at least one execution is guaranteed.
