Experiment 01

Introduction to Data Structures

Data Structures & Algorithms

üéØ Aim of the Experiment

To write a program to analyze the concept and importance of data structures through basic array operations including insertion, traversal, searching, and deletion.

üíª Program

C 
#include <stdio.h>

int main() {
    int arr[10], n = 5;

    // Initial Insertion
    printf("Inserting elements...\n");
    for (int i = 0; i < n; i++) {
        arr[i] = (i + 1) * 10; // 10, 20, 30, 40, 50
    }

    // Traversal
    printf("Array elements: ");
    for (int i = 0; i < n; i++) printf("%d ", arr[i]);
    printf("\n");

    // Searching
    int key = 30, found = -1;
    for (int i = 0; i < n; i++) {
        if (arr[i] == key) {
            found = i;
            break;
        }
    }
    if (found != -1) printf("Element %d found at index %d\n", key, found);
    else printf("Element not found\n");

    // Deletion
    int del = 20;
    for (int i = 0; i < n; i++) {
        if (arr[i] == del) {
            for (int j = i; j < n - 1; j++) arr[j] = arr[j + 1];
            n--; // reduce size
            break;
        }
    }

    // Traversal after deletion
    printf("Array after deleting %d: ", del);
    for (int i = 0; i < n; i++) printf("%d ", arr[i]);
    printf("\n");

    return 0;
}

üìã Sample Output

Output 
Inserting elements...
Array elements: 10 20 30 40 50 
Element 30 found at index 2
Array after deleting 20: 10 30 40 50

üìö Key Concepts Demonstrated

  • Data Structure: Array as a fundamental linear data structure
  • Insertion: Adding elements to the array in sequential order
  • Traversal: Accessing and displaying all elements in the array
  • Searching: Linear search to find a specific element
  • Deletion: Removing an element and shifting remaining elements
