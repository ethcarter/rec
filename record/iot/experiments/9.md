### **Experiment 09**
> *Ambient Light Sensor Using Photo Resistor and LED Lights*

---
---

#### **Aim of the experiment:**
> A program to read the ambient light using a photoresistor and indicate light levels using two LEDs.
---

#### **Components Required:**
- Resistor 100 ohm- 2
- Breadboard (generic)-1
- Photo resistor-1
- Jumper wires (generic)-1
- LED (generic)-2
- Resistor 1k ohm-1
- Arduino UNO-1


---


#### **Arduino Code:**
```ino
int light = 0; 

void setup() {
    
    Serial.begin(9600); //configure  serial to talk to computer

    pinMode(A0, INPUT); // Set photoresistor pin as input
    pinMode(13, OUTPUT); 
    pinMode(12, OUTPUT); 
}
void  loop() {

    // put your main code here, to run repeatedly:

    light = analogRead(A0);  // read and save value from PR

    Serial.println(light); // print current  light value

    if(light > 450) { // If it is bright...
        Serial.println("It  is quite light!");
        digitalWrite(13,LOW); 
        digitalWrite(12,LOW);  
    }
    else if(light > 229 && light < 451) { // If  it is average light...
        Serial.println("It is average light!");
       digitalWrite(13, HIGH);
       digitalWrite(12,LOW);  
    }
    else { // If it's dark...
        Serial.println("It  is pretty dark!");
        digitalWrite(13,HIGH); 
        digitalWrite(12,HIGH);  
    }

    delay(1000); 
}

```

---

#### **Conclusion:**
> In the above experiment, the photoresistor reliably detected ambient light and toggled LEDs across three ranges. Calibrate thresholds per environment.
