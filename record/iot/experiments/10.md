### **Experiment 10**
> *Connecting LM35 Temperature Sensor with Arduino UNO*

---
---

#### **Aim of the experiment:**
> A program to read the temperature from the LM35 sensor and display it on the Serial Monitor.

---

#### **Components Required:**
- Arduino UNO board
- LM35 Temperature Sensor

---

#### **Diagram -**
<p align="center">
<img src="9.png" alt="Circuit Diagram" width="500"/>
</p>
---

#### **Arduino Code:**
```ino

#define sensorPin A0 // LM35 is connected to this PIN

void setup() {
  
  Serial.begin(9600);
}

void loop() {

  int adcData = analogRead(sensorPin); //Read raw ADC Data
  
  float voltage = adcData * (5.0 / 1024.0);// Convert that ADC Data into voltage

  
  float temperature = voltage * 100; // Convert the voltage into temperature
  
  Serial.print("Temperature: ");
  Serial.print(temperature);
  Serial.println("*C");

  delay(800); 

}
```

---

#### **Conclusion:**
> In the above experiment, we successfully interfaced the LM35 temperature sensor with the Arduino UNO and read temperature values displayed on the Serial Monitor.
